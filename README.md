# Copilot SDLC Spec Demo

Demo mínima con GitHub Copilot SDK para convertir una idea de negocio en una especificación SDLC estructurada, validarla y guardar una traza auditable.

## Objetivo

La demo muestra tres capacidades:

1. Conversación con Copilot SDK.
2. Tools locales invocadas por Copilot.
3. Registro mínimo de trazabilidad.

## Ejecutar

```bash
python -m sdlc_demo.main
